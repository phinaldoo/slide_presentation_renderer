Slide Renderer

Service is running. Use POST /api/render (or /api/v1/render) to convert HTML slides to a ZIP bundle.

Authentication is required via X-API-Key or Authorization: Bearer <key>.

The ZIP response contains the rendered .pptx file and per-slide PNG images under slides/.

Request payload fields:

  • html (required)
  • input_files (optional list of {"file_name","base64_content"})

The renderer engine is selected by the server. Set BETA=true to use v2; otherwise the service uses v1.

Example:

{
  "html": "<div class='slide'>Hello</div>",
  "input_files": [
    {
      "file_name": "logo.png",
      "base64_content": "iVBORw0KGgoAAAANSUhEUgAA..."
    }
  ]
}
